# Pràctica Terraform: Infraestructura AWS bàsica

## Descripció
Aquesta pràctica crea una infraestructura bàsica a AWS utilitzant **Terraform**, incloent-hi VPC, subxarxes públiques i privades, instàncies EC2 i, opcionalment, un bucket S3.  

L’objectiu és entendre l’estructura modular de Terraform, l’ús de variables, condicions i sortides (outputs).
